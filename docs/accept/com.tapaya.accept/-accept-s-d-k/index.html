AcceptSDK

object AcceptSDK

The primary entry point for interacting with the Tapaya Accept SDK.

This object manages the lifecycle, initialization, and configuration of the terminal. It provides access to specialized terminal instances for different payment methods, including Card, SEPA Instant, Certis Instant, and Crypto payments.

Use initialize to set up the SDK environment, then authenticate to sign in the merchant.
