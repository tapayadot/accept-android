authenticate

fun authenticate(token: String, onSuccess: () -> Unit = {}, onError: (AcceptSDKException) -> Unit = {})

Authenticates a merchant using the provided token.

The SDK must be initialized before calling this method. On success, the SDK status transitions to AcceptSDKStatus.LOGGED_IN.

Parameters

token

The merchant authentication token obtained from your backend.

onSuccess

Called when authentication completes successfully.

onError

Called with an AcceptSDKException if authentication fails.

Throws

AcceptSDKUninitializedException

if initialize has not been called.
