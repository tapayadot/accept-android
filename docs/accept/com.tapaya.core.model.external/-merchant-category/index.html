MerchantCategory

enum MerchantCategory : Parcelable, Enum<MerchantCategory>

Represents standard Merchant Category Codes (MCC) and their associated classifications.

Each entry maps to a 4-digit ISO 18245 code used to classify businesses by the type of goods or services they provide.

Note: code is a String rather than Int to preserve leading zeros (e.g., VETERINARY with code "0742").
